Tentang Aplikasi

SmartTranslate NLP adalah sistem penerjemahan kalimat Bahasa Indonesia ke Bahasa Inggris berbasis kecerdasan buatan dan pemrosesan bahasa alami (NLP) modern. Proyek ini dibangun sebagai bagian dari Ujian Akhir Semester (UAS) Natural Language Processing.

Tujuan utama aplikasi ini adalah menyediakan antarmuka penerjemahan yang cepat, andal, dan transparan, di mana pengguna dapat melihat proses di balik layar (preprocessing dan tokenisasi) serta mengukur kualitas hasil terjemahan secara objektif dengan metrik formal.

Teknologi

Sistem ini dibangun dengan mengintegrasikan beberapa framework dan pustaka berikut:

Komponen Teknologi yang Digunakan
Backend Framework Flask 3 (Python 3.12+)
Frontend HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
Deep Learning Framework PyTorch, HuggingFace Transformers
Pre-trained Model MarianMT (Helsinki-NLP/opus-mt-id-en)
Analisis Data Pandas, NumPy
NLP Helpers NLTK (Natural Language Toolkit)
Evaluasi Kinerja SacreBLEU, ROUGE Score, NLTK Meteor
Database Penyimpanan SQLite (smarttranslate.db)

Alur Sistem

Proses penerjemahan dari input pengguna hingga hasil akhir melewati tahapan berikut:

1. Input Kalimat
Pengguna memasukkan kalimat dalam Bahasa Indonesia melalui antarmuka web.
2. Preprocessing & Normalisasi
Teks diubah menjadi huruf kecil, karakter non-alphanumeric dibersihkan, dan whitespace dinormalisasi.
3. Tokenisasi
Teks dibagi menjadi satuan token kata dasar menggunakan NLTK sebelum dikirim ke model neural net.
4. Transformer MarianMT (Decoder-Encoder)
Token diproses oleh model Helsinki-NLP/opus-mt-id-en untuk menghasilkan representasi semantik bahasa Inggris.
5. Evaluasi Kinerja
Kalimat terjemahan dibandingkan dengan dataset referensi untuk menghitung BLEU, ROUGE, dan METEOR.
6. Output & Riwayat
Hasil terjemahan ditampilkan di layar dan disimpan secara otomatis ke database SQLite.

Model NLP

Aplikasi menggunakan model MarianMT yang disediakan oleh Universitas Helsinki (Helsinki-NLP) di platform Hugging Face dengan nama direktori Helsinki-NLP/opus-mt-id-en.

Model ini dilatih menggunakan korpus paralel besar OPUS. Menggunakan arsitektur Transformer standar (Encoder-Decoder) yang sangat efisien untuk tugas penerjemahan mesin dua arah.

Instalasi & Cara Menjalankan

Ikuti langkah-langkah berikut untuk menjalankan aplikasi di lingkungan lokal Anda:

1. Kebutuhan Sistem
  • Python 3.12 atau versi di atasnya
  • Pip (Python Package Installer)
  • Koneksi internet aktif untuk mengunduh model pada saat pertama kali aplikasi dijalankan
2. Instalasi Dependensi
pip install -r requirements.txt
3. Menjalankan Aplikasi
flask run

Setelah dijalankan, buka peramban web dan akses halaman di alamat http://127.0.0.1:5000.

Kelebihan & Kekurangan

Kelebihan
  • Akurasi penerjemahan tinggi berkat arsitektur deep learning Transformer.
  • Kecepatan eksekusi cepat menggunakan cache lokal model.
  • Tersedia visualisasi hasil evaluasi langsung.
  • Dukungan preprocessing yang lengkap dan transparan.
Kekurangan
  • Memerlukan memori RAM yang cukup besar saat memuat model pertama kali.
  • Model sangat terikat pada domain data latih OPUS.
  • Ketergantungan pada pustaka eksternal yang cukup banyak.

Lisensi

Proyek SmartTranslate NLP dilisensikan di bawah Lisensi MIT. Anda dibebaskan untuk memodifikasi dan membagikan kembali proyek ini untuk keperluan akademis maupun non-komersial.
